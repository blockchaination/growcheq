# GrowCheq

The all-in-one customer engagement platform for UK businesses.

## Project Info

GrowCheq helps businesses turn every interaction into revenue by unifying reviews, messaging, payments, and more into a single platform.

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

This project is configured for deployment on Vercel or similar platforms.
